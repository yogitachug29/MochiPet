# MochiPet - Installation Guide

## Quick Start

1. **Download** `MochiPet.zip`
2. **Extract** it to any folder (e.g., Desktop)
3. **Run** `MochiPet.exe`
4. Done! 🎉

## Auto-Updates

MochiPet automatically checks for updates when you run it. If a new version is available, it will:
- Download the update
- Install it
- Restart automatically

You don't need to download anything manually!

## Troubleshooting

### "Windows protected your PC" message?
This is normal for apps not yet widely distributed. Click:
1. "More info"
2. "Run anyway"

After a few downloads, Windows will stop showing this warning.

### App doesn't update?
- Make sure you have internet connection
- Update check happens in background on startup
- Check for new version availability on GitHub: https://github.com/yogitachug29/MochiPet/releases

## Where to get the latest version

Visit: https://github.com/yogitachug29/MochiPet/releases

Download the latest `MochiPet.zip` file.
